# 파이썬 Docstring 규칙

파이썬(`*.py`) 파일을 작성하거나 수정할 때는 반드시 파일의 맨 상단에 해당 파일의 목적과 내용을 설명하는 docstring을 작성해야 합니다.

## 규칙 상세
1. **위치**: 파일의 가장 첫 번째 줄(shebang 라인이 있다면 그 다음)에 위치해야 합니다.
2. **형식**: 삼중 따옴표(`""" ... """`)를 사용합니다.
3. **내용**: 
   - 파일의 전반적인 역할 요약
   - 주요 기능 및 모듈 설명
   - 한국어로 작성
