# 📂 ICB 10기 2조 통합 프로젝트

이 저장소는 ICB 10기 2조의 프로젝트 결과물을 관리하는 공간입니다.  
본 프로젝트는 **iM뱅크 PB를 위한 Money Move 방어 대시보드 v2**와 **네이버 API 연동 어플리케이션** 두 개의 독립된 서브 프로젝트로 구성되어 있습니다.

---

## 🔗 프로젝트 바로가기 (Sub-Projects)

각 서브 프로젝트의 구체적인 구현 설명, 환경 설정 방법, 진행 단계별 체크리스트는 아래의 개별 `README.md` 링크를 통해 확인하실 수 있습니다.

### 1. [🏦 iM뱅크 PB Money Move 방어 대시보드 v2](./bank%20ai/README.md)
- **목적:** PB 고객들의 자산 유출(Money Move) 위험을 실시간으로 감지하고 방어하기 위한 대시보드 앱.
- **주요 기능:** 실시간 리스크 스코어링, 방어 성공 시뮬레이션(Session State), 맞춤형 추천 상품 및 CRM Playbook 대본 제공.
- **배포 주소:** [Streamlit 서비스 바로가기](https://icb10project2-mg6qetkpk9xj88e6khuekr.streamlit.app/)

### 2. [🔍 네이버 API 연동 어플리케이션](./naver-api-app/README.md)
- **목적:** 네이버 검색 API와 OpenAI API를 활용하여 트렌드 데이터를 수집하고 요약 보고서를 자동화하는 모듈.
- **주요 기능:** 네이버 블로그/뉴스 검색 데이터 수집, LLM 기반 스마트 요약 보고서 파일 생성 및 자동 저장.

---

## 🛠️ 공통 인프라 & 자동화 스크립트

- **변경 사항 감시 및 자동화 스크립트 ([watch_changes.ps1](./watch_changes.ps1))**:
  - 파워쉘 기반의 실시간 코드 변경 사항 감시 툴입니다. 
  - 로컬 환경에서 소스코드의 수정이 일어날 때마다 자동으로 Git 스테이징 및 커밋을 수행합니다.
- **로컬 가상환경 설정 (`.venv`)**:
  - 프로젝트는 가상환경 매니저인 `uv`를 사용하며, 루트 폴더의 `.venv` 가상환경이 생성되어 동작합니다.
  - 공통 의존성은 [requirements.txt](./requirements.txt)를 참고하세요.
